\# SmartVis



SmartVis is a real-time audio visualizer for Spicetify. It uses DOM-based bars to display the audio waveform and responds to BPM, volume, and pump modes.



\## Features

\- \*\*BPM Tap Sync:\*\* Tap to sync the visualizer to your music.

\- \*\*Pump Modes:\*\* Switch between `pump`, `pulse`, `pulse-pump`, and `off`.

\- \*\*Custom Colors:\*\* Cycle through predefined colors or use Spotify's theme color.

\- \*\*Lightweight:\*\* Optimized with minimal DOM and CSS updates for smooth performance.

\- \*\*Adaptive Brightness:\*\* Bars respond dynamically to volume and beat intensity.



\## Installation

```bash

spicetify config extensions smartvis.js
